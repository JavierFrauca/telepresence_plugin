# README - Documentación Técnica Telepresence GUI

Este directorio contiene la documentación técnica de la extensión Telepresence GUI para VS Code.

## Contenido

- [Anatomía de una Extensión de VS Code](./anatomia_extension_vscode.md) - Explica los componentes principales que conforman una extensión de VS Code, usando Telepresence GUI como ejemplo.
- [Documentación Técnica de Telepresence GUI](./documentacion_tecnica_telepresence.md) - Detalle técnico completo de la arquitectura, componentes y funcionamiento de la extensión.

## Propósito

Esta documentación tiene dos objetivos principales:

1. Proporcionar una visión general de cómo se estructura una extensión de VS Code, útil para desarrolladores que quieran aprender a crear extensiones.
2. Ofrecer una documentación técnica detallada sobre el funcionamiento interno de la extensión Telepresence GUI.

## Uso

Los documentos están en formato Markdown y pueden visualizarse directamente en VS Code o cualquier otro visor de Markdown.

Para los desarrolladores interesados en contribuir al proyecto, se recomienda revisar primero el documento de anatomía general y luego profundizar en la documentación técnica específica.
